# brew 없이 맥에서 thrift 설치하기

* 설치 순서는 [Apache Thrift OS X Setup](https://thrift.apache.org/docs/install/os_x)에서 따라 진행
* bison 버전이 낮을 경우 최신 버전 설치

``` shell
brew install bison
brew link bison --force
```
